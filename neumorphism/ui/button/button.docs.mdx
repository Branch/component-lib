## React Component for showing a button

A button with some text and a click handler

### Component usage
```js
<Button text="hello from Button" type="convex" size="" clickHandler={() => {alert('clicked')}} padding={[20, 10, 20, 10]} />
```

### Using props to customize the button

Modify the text to see it change live:
```js live
<Button text="hello from Button" type="convex" size="" clickHandler={() => {alert('clicked')}} padding={[20, 10, 20, 10]} />
```
