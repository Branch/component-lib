## React Component for rendering text

A basic link that renders some text

### Component usage
```js
<Link text="hello from Link" url="https://bit.dev" type="flat" padding={[10, 20, 10, 20]} size={""} newWindow={true} />
```

### Using props to customize the text

Modify the text to see it change live:
