## Card component

- renders a Heading Component
- renders a Text Component
- renders an External Link

### Component usage

Render component with any heading, text and link. Available types are:
 - flat
 - convex
 - concave
 - pressed

Optional border radius sizes:
 - sm
 - md
 - lg

Padding in pixels

```js
<Card
    heading="Self Host"
    text="lean how to self host these components"
    type="flat"
    size=""
    padding={[10,10,10,10]}
    image={"https://gpresults.info/static/media/fabio-tinted.c64f6f24.png"}
    tags={['React', 'Express']}
  />
```
