## Card component

- renders a Heading Component
- renders a Text Component
- renders an External Link

### Component usage

Render component with any heading, text and link. Available types are:
 - flat
 - convex
 - concave
 - pressed

Optional border radius sizes:
 - sm
 - md
 - lg

Padding in pixels

```js
<Card
  link="https://harmony-docs.bit.dev/aspects/generator"
  heading="My projects"
  text="to see a list of my projects"
  type="flat"
  size="sm"
  padding={[10,10,10,10]}
/>
```
